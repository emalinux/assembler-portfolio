Assembler Studio

Sito portfolio e identità digitale di Assembler Computer. Progetto tecnico, UX-oriented e completamente personalizzato.

🧠 Descrizione

Assembler Studio è il sito portfolio ufficiale di Assembler Computer, pensato per raccontare e valorizzare i progetti web realizzati nel tempo, con un approccio tecnico, essenziale e orientato all’esperienza utente.

Non è un semplice sito vetrina, ma un progetto strutturato che riflette il metodo di lavoro Assembler:
chiarezza, controllo totale del codice, prestazioni elevate e design funzionale.

Il sito è stato sviluppato interamente con Hugo, senza l’uso di temi preconfezionati, per garantire massima libertà progettuale e una base solida, riutilizzabile e scalabile.

🚀 Tecnologie

  • Hugo – Static site generator ad alte prestazioni
  • HTML & CSS personalizzato – Layout pulito, coerente e responsive
  • JavaScript vanilla – Interazioni leggere e controllate
  • Sistema multilingua (i18n) – IT / EN per rafforzare la percezione professionale
  • Netlify – Deploy continuo da GitHub
  • GitHub – Versionamento completo del progetto
