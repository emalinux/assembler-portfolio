Bed & Breakfast A Me Cà

A cozy B&B in Albisola Marina. Simple and mobile-first website built for fast access and a relaxed user experience.

🛏️ Description

Official website of Bed & Breakfast A Me Cà, a charming family-run guesthouse located in Albisola Marina, just a few steps from the sea.

Designed with a mobile-first approach, the website was built using Hugo and custom HTML/CSS, offering a simple yet refined user experience — ideal for all age groups, including less tech-savvy users.

The layout is minimalist yet warm, with a palette inspired by the Ligurian coast.
It includes essential sections: introduction, photo gallery, contacts, and booking options.
Optimized for fast loading even on mobile networks.

🚀 Technologies

  • Hugo (static site generator)
  • Custom HTML & CSS for a tailored, welcoming design
  • Netlify for fast deployment and free hosting
  • SEO optimization for local search (Ligurian B&Bs)
  • Responsive layout for smartphones and tablets
