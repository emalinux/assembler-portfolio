# Assembler Studio – Web & UX Development

Sito portfolio ufficiale di Assembler Studio.

Progetto statico sviluppato con Hugo Extended, layout personalizzato e architettura modulare orientata a UX, performance e chiarezza strutturale.

[![Netlify Status](https://api.netlify.com/api/v1/badges/17d2e0a9-c277-4664-b9c7-b688a2377c72/deploy-status)](https://app.netlify.com/projects/assembler-studio/deploys)


---

## 🎯 Visione

Assembler Studio nasce come estensione digitale del laboratorio Assembler Computer.

Se il laboratorio ripara macchine,
lo Studio progetta sistemi digitali.

Obiettivo:
- Siti statici performanti
- UX pulita e leggibile
- Architettura modulare
- Design tecnico, non decorativo

---

## 🔧 Stack Tecnologico

- Hugo v0.147.9 (extended)
- HTML5 semantico
- CSS custom (design system interno)
- JavaScript vanilla
- Filtri dinamici per progetti
- Deploy su Netlify
- Versionamento GitHub

---

## 🧠 Architettura

Struttura modulare:

```plaintext
archetypes/
content/
data/
i18n/
layouts/
static/
hugo.toml
```
---

Partial dedicati:
- header
- hero
- project cards
- modals
- footer
- filtro per tag
- toggle tema

Design system interno con:
- Variabili coerenti
- Struttura tipografica modulare
- Separazione sezioni chiara

---

## 🚀 Deploy

Deploy automatico tramite Netlify collegato al branch `main`.

Build command:

hugo –gc –minify
