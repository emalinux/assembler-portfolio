## 🛠️ Overview

Development of the official website for **Shiatsu School Cuneo**, a training center focused on Shiatsu practice and education.

The goal of the project was clear:  
to create an online presence **aligned with the nature of the discipline**, avoiding unnecessary visual noise or over-designed elements.

The website was designed to convey:

- balance  
- clarity  
- authenticity  
- trust  

Built entirely with **Hugo**, the project avoids traditional CMS solutions in order to ensure:

- high performance  
- long-term stability  
- simple maintenance  
- full control over the codebase  

Special attention was given to:

- content readability  
- clear structure of training paths  
- clean visual hierarchy  
- consistency between design and message  
- mobile experience  

The result is a minimal, fast and solid website, designed to guide users naturally without distractions.

---

## 🚀 Technologies

- **Hugo** – high-performance static site generator  
- Custom **HTML & CSS**  
- Mobile-first responsive design  
- Basic SEO optimization  
- Automated deploy via Git + Netlify  
- No database, no CMS
