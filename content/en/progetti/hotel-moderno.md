## 🛠️ Overview

Development of the official website for **Hotel Moderno Demonte**, a mountain hotel located in the heart of Valle Stura (Northern Italy).

The goal of the project was to create a digital presence that feels:

- welcoming  
- authentic  
- connected to the local territory  

The website was built from scratch using **Hugo**, avoiding traditional CMS platforms in order to ensure:

- fast loading times  
- a lightweight and stable structure  
- long-term maintainability  
- full control over the codebase  

Special attention was given to:

- clear and intuitive navigation  
- content readability  
- visual storytelling through images  
- mobile-first optimization  

The result is a clean, fast and conversion-oriented website, designed for real users rather than unnecessary technical complexity.

[🌐 Visit website →](https://hoteldemonte.com/)

## 🚀 Technologies

- **Hugo** – high-performance static site generator  
- **Custom HTML & CSS**  
- Responsive mobile-first design  
- Basic SEO optimization  
- Automated deploy via Git + Netlify  
- No database, no CMS
