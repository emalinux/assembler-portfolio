## 🛠️ Description

Development of the official website for **Paola Bovero**, teacher, author, and a figure deeply connected to a personal and spiritual journey.

The project was designed to create a digital space aligned with the author's identity:  
intimate, narrative, and elegant.

The website was built from scratch using **Hugo**, deliberately avoiding traditional CMS platforms to ensure:

- high performance
- full control over the codebase
- no unnecessary complexity
- long-term stability

Special attention was given to:

- typography and reading rhythm
- narrative section structure
- balance between content and imagery
- a consistent visual atmosphere

The result is a clean, readable, and strongly identity-driven editorial website.

[🌐 Visit the website →](https://paolabovero.com/)

---

## 🚀 Technologies

- **Hugo** – ultra-fast static site generator
- **Custom HTML & CSS**
- Responsive editorial layout
- SEO optimization
- Automated deployment via Git + Netlify
- No database, no CMS
